C2C

Propuesta C2C

Propuesta para generar un c2c diferenciado, que ayudara al usuario a seleccionar según el producto que necesitaban adquirir (Móvil/Hogar)

Descubrimiento e Investigación

Este requirimiento nació porque anteriormente no existía la opción de seleccionar el tipo de producto que el usuario estaba adquiriendo, y habían 2 links por separado; el cliente solicitó una propuesta para poder unirlos en uno solo.

Se inició con breve benchmark, para comprobar si otras compañías utilizaban esta opción de c2c; luego se procedió a realizar las propuestas.
